# Login Flutter

![Vista Login](assets/imagenes/emulador.png)

Este proyecto consiste en una interfaz de inicio de sesión desarrollada en Flutter utilizando Dart.

La aplicación incluye:

* Fondo degradado moderno.
* Imagen de logo personalizada.
* Campo de correo electrónico.
* Campo de contraseña.
* Botón de ingreso.
* Diseño responsivo.
* Estilos personalizados.

## Tecnologías Utilizadas

* Flutter
* Dart
* Visual Studio Code
* Android Studio

## Estructura del Proyecto

El documento de la interfaz se encuentra en `lib/main.dart`.

## Campos de texto

La interfaz contiene:

* Campo de correo electrónico.
* Campo de contraseña.

Con estilos personalizados y bordes redondeados.

## Emulador

Para ejecutar y visualizar la aplicación se utilizó un emulador de Android Studio, donde se pudo comprobar el funcionamiento de la interfaz y la rotación de pantalla del dispositivo.

## Dependencias

Este proyecto utiliza únicamente Flutter SDK.

## Conclusión

Con este proyecto se logró desarrollar una interfaz moderna de login utilizando Flutter, aplicando diseño visual, manejo de widgets y personalización de componentes.

Además, se aprendió:

* Uso de widgets.
* Diseño responsivo.
* Uso de emuladores Android.
